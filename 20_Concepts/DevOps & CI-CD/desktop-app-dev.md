# Desktop app dev

**Definition**: The process of designing, building, and maintaining software applications that run natively on desktop operating systems (e.g., Windows, macOS, Linux). These applications typically leverage platform-specific APIs and frameworks to deliver performance, offline functionality, and deep integration with the host OS.

**Key Details**:
- **Platforms**: Targets Windows (Win32, UWP), macOS (Cocoa, SwiftUI), or Linux (GTK, Qt).
- **Technologies**: Uses languages like C++, C#, Swift, or frameworks like Electron (for cross-platform web-based apps).
- **Distribution**: Often packaged as installers (e.g., `.exe`, `.dmg`, `.deb`) or via app stores.
- **Characteristics**: Prioritizes performance, local resource access, and OS-specific UI/UX conventions.
