# Which

A command-line utility in Unix-like operating systems used to locate executable files in the user's PATH. It is commonly employed in desktop app development to verify the presence and location of required tools or dependencies before execution.

**Key Details:**
- **Purpose:** Resolves the full path of a command or executable.
- **Usage:** `which [command]` (e.g., `which python`).
- **Output:** Returns the absolute path of the executable or an error if not found.
- **Relevance in Desktop App Dev:** Ensures build scripts or automation tools can locate necessary binaries (e.g., compilers, package managers).
